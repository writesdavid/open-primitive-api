## 3. Protocol Landscape

### WebMCP (W3C)

Major development. Google, Microsoft, and Anthropic jointly submitted the Web Model Context Protocol Specification v1.0 to the W3C Web Applications Working Group in January 2026. Released as a W3C Draft Community Group Report on February 10, 2026.

**Chrome 146 Canary** ships WebMCP in early preview. The `navigator.modelContext` API lets web pages declare structured tools with defined schemas, parameters, and security boundaries.

**Timeline:** Candidate Recommendation targeted Q3 2026. Safari and Firefox support expected late 2026 / early 2027. V2.0 targeting late 2027 with streaming responses, tool composition, and decentralized discovery.

**89% token efficiency improvement** over screenshot-based methods. This is relevant for Open Primitive -- if we expose WebMCP metadata on our tool pages, agents can interact with our data natively in the browser.

### A2A Protocol (Google)

Version 0.3 released. Adds gRPC support, security card signing, extended Python SDK client support. Google adding A2A to Agent Engine for production-ready agent deployment. 150+ organizations in the ecosystem.

A2A also moved to Linux Foundation governance.

**Assessment:** A2A is agent-to-agent; MCP is agent-to-tool. They're complementary. Open Primitive is a tool/data provider, so MCP and WebMCP are our protocols. A2A matters if agents start brokering our data to other agents.

## 4. Regulatory

- March 17: European Data Protection Supervisor published the EDPS Compass defining its supervisory role under the AI Act for 2026-2027.

### US AI Policy

No comprehensive federal AI law. Patchwork of state laws and agency guidance. Trump's December 2025 Executive Order signals intent to consolidate AI oversight but establishes no standards or regulations on its own.

**Colorado AI Act** implementation postponed from February 1 to June 30, 2026.

### Federal Data Deletions

The situation remains severe and unresolved:
- ~8,000 web pages and ~3,000 datasets removed or modified since January 2025, primarily affecting DEI, gender identity, public health, environmental policy, and social programs.
- Two individuals accused of deleting 96 government databases, many containing FOIA records and investigative files.
- Some CDC/FDA/HHS pages restored under court order (February 2025 restraining order from Doctors for America lawsuit).
- No comprehensive accounting of what has been permanently lost.
- Librarians and archivists running preservation efforts nationwide.

**Opportunity signal:** The gap between deleted federal data and public need is exactly Open Primitive's territory. Every deleted dataset is a potential tool.

## 6. Cloudflare

### x402 Foundation

Coinbase and Cloudflare forming the x402 Foundation. x402 is now integrated into Cloudflare's Agents SDK and MCP Servers. Developers can build agents that automatically make payments using x402, and services can return 402 Payment Required responses.

### x402 v2 upgrades (early 2026)

- Wallet-based identity
- Automatic API discovery
- Dynamic payment recipients
- Multi-chain and fiat support via CAIP standards
- Modular SDK for custom networks and payment schemes

### World AgentKit (March 17, 2026)

Sam Altman's World project launched AgentKit -- cryptographic proof that an AI agent is backed by a verified human identity. Built as an extension to x402 v2, in coordination with Coinbase. Any website using x402 can enable proof of unique human verification alongside micropayments.

**Assessment:** x402 + AgentKit creates an infrastructure where agents can pay for data and prove human backing. Open Primitive could charge agents for premium data access via x402 on Cloudflare Workers. This is a monetization path.

## Strategic Summary

Three things happened this week that matter for Open Primitive:

**1. The federal government is shipping its own MCP servers.** GovInfo and Census Bureau have official MCP servers. CMS and Treasury are building them. This validates the market but means raw API access is becoming commoditized. Open Primitive must differentiate on the human layer -- readability, cross-referencing, editorial judgment about what matters.

**2. WebMCP is real and shipping in Chrome.** Google, Microsoft, and Anthropic submitted a W3C spec. Chrome 146 Canary has it. Every website can become a structured tool for agents. Open Primitive should be among the first government data sites to implement WebMCP metadata. This is a concrete technical priority.

**3. Block is reorganizing around AI.** 40% workforce cut. AAIF co-founding. Cash App Moneybot. Square AI. The company David works at is making the largest AI bet in fintech. Open Primitive's positioning at the intersection of verified data and agent infrastructure aligns with where Block is heading.

**Immediate action items:**
- Investigate WebMCP implementation for Open Primitive tools
- Evaluate building an Open Primitive MCP server (the lzinga project proves single-developer MCP servers can reach 300+ tools)
- Monitor CMS MCP server development -- direct competitor to Hospitals tool
- Consider x402 integration on Cloudflare Workers for agent-payable data access
