Ranked by impact. The COO should propose these in order.

### 1. Ship /v1/history retrieval (highest leverage)
The archive middleware is already writing data. Build the read side. This turns a cost center into the moat. Every day without retrieval is archived data nobody can query.

OPP's core claim is cryptographic provenance. If the signing key isn't set in Vercel env vars, every response is unsigned. One `curl` confirms or denies this.
