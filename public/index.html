The Envelope

Every response carries proof

OPP defines a standard envelope. Every domain, every endpoint, every response ships the same metadata. Agents parse it once and trust it everywhere.

  • domain — Which dataset answered the query
  • source — The federal agency that produced the data
  • source_url — Direct link to the upstream record
  • freshness — ISO 8601 timestamp of last data pull
  • confidence — 0–1 score of data completeness
  • citations — Array of source references agents can paste directly
  • proof — Hash of the upstream data at query time

Agent-Native

Five ways in. Zero friction.

  • /v1/ask?q= — Natural language. You ask, we route to the right domain. curl -H "X-API-Key: KEY" "https://api.openprimitive.com/v1/ask?q=is+austin+tap+water+safe"
  • MCP Server — 13 tools. One command. npx @openprimitive/mcp-server --api-key YOUR_KEY
  • A2A Agent Card — 13 skills at /.well-known/agent.json Google A2A compatible. Point any orchestrator at the card and go.
  • Alert Feed — Subscribe to new data across any domain. Webhook or SSE. Get notified when upstream agencies publish new records.
  • Citation-Ready — Every response includes formatted citations agents paste directly. No post-processing. No hallucinated sources. The citation array is copy-paste ready.
