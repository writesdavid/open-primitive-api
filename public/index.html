7 Domains

Flights

FAA NAS + Open-Meteo

Live delays and hub weather for 8 US carriers.

curl -H "X-API-Key: KEY" https://api.openprimitive.com/v1/flights/DL

Cars

NHTSA

Crash safety ratings and recalls for any US vehicle.

curl "https://api.openprimitive.com/v1/cars?year=2024&make=Toyota&model=Camry&api_key=KEY"

Food

FDA Enforcement

Active food recalls from the last 90 days.

curl -H "X-API-Key: KEY" "https://api.openprimitive.com/v1/food?q=peanut+butter"

Water

EPA SDWIS

Drinking water systems and violations by ZIP code.

curl -H "X-API-Key: KEY" "https://api.openprimitive.com/v1/water?zip=10001"

Drugs

FDA FAERS

Adverse events, top reactions, and label warnings.

curl -H "X-API-Key: KEY" "https://api.openprimitive.com/v1/drugs?name=ibuprofen"

Hospitals

CMS Care Compare

Quality ratings, mortality, readmissions, patient experience.

curl -H "X-API-Key: KEY" "https://api.openprimitive.com/v1/hospitals?q=mayo+clinic"

Health

PubMed / MEDLINE

Research evidence for supplements and health claims.

curl -H "X-API-Key: KEY" "https://api.openprimitive.com/v1/health?q=vitamin+d"

For AI Agents

Built for the machine layer

Open Primitive ships as an MCP server, exposes an A2A-compatible agent card, and publishes llms.txt for LLM discovery. Point any agent framework at it and go.

MCP Server Config

// claude_desktop_config.json { "mcpServers": { "open-primitive": { "command": "node", "args": ["/path/to/open-primitive-api/mcp.js"] } } }

7 tools registered: get-flights, get-car-safety, get-food-recalls, get-water-safety, get-drug-safety, get-hospital-quality, get-health-evidence.

A2A agent card at /.well-known/agent.json. OpenAPI spec at /openapi.json.

Every Response

One envelope, every domain

Agents parse it once. Every endpoint returns the same four metadata fields alongside domain-specific data.

  • domain — Which dataset ("flights", "cars", "water"...)
  • source — The federal agency or data provider
  • source_url — Direct link to the upstream source
  • freshness — ISO 8601 timestamp of last data pull
{ "domain": "flights", "source": "FAA NAS + Open-Meteo", "source_url": "https://nasstatus.faa.gov", "freshness": "2026-03-19T14:30:00Z", // ... domain data }

Get Started

Three steps. Under a minute.

  1. Sign up at openprimitive.com/pricing and grab your API key.
  2. Pass the key as an X-API-Key header or api_key query parameter.
  3. Hit any of the 7 endpoints. That's it.
# Check flight delays for Delta curl -H "X-API-Key: YOUR_KEY" https://api.openprimitive.com/v1/flights/DL # Search hospital quality curl -H "X-API-Key: YOUR_KEY" "https://api.openprimitive.com/v1/hospitals?q=mayo+clinic" # Or use a query parameter curl "https://api.openprimitive.com/v1/drugs?name=metformin&api_key=YOUR_KEY"
