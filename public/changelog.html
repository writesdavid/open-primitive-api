Changelog

Build history for the Open Primitive API.

March 19, 2026
Launch
  • 13 federal data domains live: flights, cars, food, water, drugs, hospitals, health, nutrition, jobs, demographics, products, SEC, safety
  • Cross-domain safety score: composite water + hospital quality per ZIP code
  • Location profile: demographics + safety in one call
  • Compare endpoint: side-by-side comparison of ZIPs, drugs, or hospitals
  • MCP server with 13 tools for Claude, Cursor, and MCP-compatible agents
  • A2A agent card with 13 skills for agent-to-agent discovery
  • Agent traffic detection and real-time stats dashboard
  • Historical data archiving (time-series starts now)
  • Webhook alerting for new recalls and violations
  • API key self-service registration
  • System status page monitoring all 12 upstream federal APIs
  • OpenAPI 3.1 spec, llms.txt, llms-full.txt for agent discovery
  • Data quality grades on every response
  • 10 federal agencies connected: FAA, NHTSA, FDA, EPA, CMS, NIH/NLM, USDA, BLS, Census Bureau, SEC
