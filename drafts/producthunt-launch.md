# Product Hunt Launch Kit — Open Primitive

## Tagline (60 chars max)

17 US federal data APIs. One key. Built for AI agents.

## Description (260 chars max)

Structured government data from 11 federal agencies. FDA drug safety, EPA water quality, CMS hospital ratings, and 14 more domains. Every response signed with Ed25519 provenance. MCP server included.

## First Comment (Maker Comment)

Federal data sits across 11 agencies, each with its own format, its own quirks, its own breaking changes. If you build agents that need government data, you already know the pain.

Open Primitive normalizes 17 domains into one API. FDA drug safety, EPA water quality, NHTSA recall data, CMS hospital ratings — all structured the same way, all through one key.

Every response comes with Ed25519-signed provenance. Your agent can verify the data actually came from the source it claims. No trust required.

Free. No API key. No limits. MCP server ships out of the box for Claude and Cursor.

We built this because agents deserve first-class access to public data. Tell us what federal domain to add next.

— the Open Primitive team
