## PR Title

Add writesdavid/open-primitive-api

## PR Description

Open Primitive is an MCP server that exposes 13 US federal data domains as tools. It pulls from FAA, NHTSA, FDA, EPA, CMS, NIH, USDA, BLS, Census, and SEC — covering flights, cars, food, water, drugs, hospitals, health, nutrition, jobs, demographics, consumer products, SEC filings, and cross-domain safety scores.

All data comes from primary federal sources. No scraping, no third-party aggregators.
