# GitHub Issue for mcp.so Submission

**Title:** Add Open Primitive — 17 US federal data domains as MCP tools

**Body:**

**Server Name:** open-primitive-mcp
**npm:** https://www.npmjs.com/package/open-primitive-mcp
**GitHub:** https://github.com/writesdavid/open-primitive-api
**MCP Registry:** io.github.writesdavid/open-primitive

**Description:** 17 US federal data domains (FAA, NHTSA, FDA, EPA, CMS, NIH, USDA, BLS, Census, SEC, NOAA) as MCP tools. Signed responses with Ed25519 provenance. Natural language query routing. Cross-domain risk scores, drug comparisons, location profiles.

**Install:**
```json
{ "mcpServers": { "open-primitive": { "command": "npx", "args": ["-y", "open-primitive-mcp"] } } }
```

**Tools (17):** get-flights, get-car-safety, get-food-recalls, get-water-safety, get-drug-safety, get-hospital-quality, get-health-evidence, get-nutrition, get-jobs, get-demographics, get-product-recalls, get-sec-filings, get-weather, get-location-profile, get-comparison, get-safety-profile, ask-question

**API:** https://api.openprimitive.com
**Spec:** https://openprimitive.com/protocol.html
